지루하지 않은 홈페이지를 만들 수 있습니다.

Animation

SVG Animation

SVG의 stroke-dasharray stroke-dashoffset 속성을 이용하면 선에도 애니메이션을 부여할 수 있습니다. 이러한 특징을 이용해서 텍스트 애니메이션을 구현했습니다.

Cube Animation

정육면체를 6개의 DIV로 나누어 DIV마다 회전, 크기 변화 애니메이션을 주었습니다.

Tail Animation

CSS 효과를 이용한 TAIL 애니메이션 입니다. DIV 요소를 만든 후 각각의 CLASS에 맞게 CSS효과를 적용했습니다. 또한, KEYFRAMES 이용하여 역동적인 애니메이션 효과를 주었습니다.

Dots Animation

3개의 원이 합쳐진 원 6개를 삼각형 구조로 배치 후 KEYFRAMES를 이용하여 회전 애니메이션 효과를 주었습니다.

Circle Animation

6개의 면마다 TRANSFORM ROTATE을 주어 합쳤을 때 원 모양을 나타내도록 하였으며 KEYFRAMES를 이용하여 시간이 지날 때마다 다른 각도로 회전할 수 있도록 하였습니다.
